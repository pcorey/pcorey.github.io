Want to scan another app?

Drag and drop your .meteor/versions file into the box above to find out if your application is using any packages with known vulnerabilities.

This tool is powered by east5th:package-scan and runs entirely within your browser. We don't send, save, sleuth, sniff or store your versions file in any way. If you're still feeling nervous, check out a set of sample output first.

Know about a vulnerable package that we're not reporting? Please help out the community and contribute!

Include this tool in your build process:

meteor add east5th:package-scan
